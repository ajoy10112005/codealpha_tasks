# Flashcard Quiz App

## CodeAlpha App Development Internship - Task 1

### Description
This is a simple Flashcard Quiz App developed using Kotlin and Jetpack Compose in Android Studio.

### Features
- Display flashcard questions
- Show answer on button click
- Navigate using Next button
- Navigate using Previous button
- Simple and user-friendly interface

### Technologies Used
- Kotlin
- Jetpack Compose
- Android Studio

### How to Run
1. Clone or download the project.
2. Open it in Android Studio.
3. Sync Gradle.
4. Run the app on an emulator or Android device.
